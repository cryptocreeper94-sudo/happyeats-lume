GLOBAL STYLE & DESIGN
==================================================
- Layout: 3-column Bento grid (modular, responsive)
- Navigation: Accordion dropdowns, horizontal carousels
- Effects: Glassmorphism panels, subtle 3D depth, glow CTAs
- Typography: Bold sans-serif headers, clean sans-serif body
- Color Palette:
  * Deep Navy (trust, stability)
  * Emerald Green (freshness, growth)
  * Warm Cream/Off-White (balance, readability)
  * Gold accents (premium highlights, glow effects)
  * Soft Gray (neutral background panels, glassmorphism depth)
- Principle: ≤ 4 taps per flow, predictable navigation
- Dark mode default for drivers (night use), light mode option for vendors/admins

==================================================
TRAINING MODULE – HOW TO USE THE APP
==================================================
INTRODUCTION
- Designed for truck drivers, vendors, and admins.
- Solves access issues for drivers stuck at stops/terminals.
- Connects drivers to vendors and delivery runners quickly.

STEP 1: ONBOARDING
- Guided overlay introduces core functions.

STEP 2: DRIVER FLOW
- GPS pin → Break Timer → Food/Essentials → Vendor carousel → Cart → ETA countdown → Delivery → Expense auto-log.

STEP 3: VENDOR FLOW
- Dashboard overlay → Orders accordion → Prep timers → Status updates → Revenue summary.

STEP 4: ADMIN FLOW
- Dashboard overlay → Region Control → Analytics → Reports → Expense tracking → Help modals.

STEP 5: HELP & INSTRUCTIONS
- “?” icon on every screen → modal instructions.
- Global Help Center + tooltips.

STEP 6: BUSINESS TOOLS
- Built-in PDF creator for flyers, letters, pitch decks.
- Templates for vendor outreach, driver recruitment, investor pitches.
- AI Business Agent handles invoices, expenses, payouts, communications.

STEP 7: SCALING ROADMAP
- Phase 1: Nashville pilot (small vendor network).
- Phase 2: Regional expansion (Memphis, Louisville, Atlanta).
- Phase 3: National rollout (major freight hubs).
- Phase 4: Full logistics integration.
